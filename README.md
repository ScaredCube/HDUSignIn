# HDU SignIn

**声明：本程序旨在不使用钉钉的情况下打开上课啦签到，仍然需要签到码和实时定位。使用本程序造成的一切影响由使用者本人承担**

主要是杭电钉加载实在是有点慢了，课室人多或者网不好的环境下经常打不开。故写了个单独的app用于优雅地上课签到。使用安卓原生开发保证app的小巧体积和流畅度，没有ios版本。

## 使用说明

### 初次使用

点击`登录` -> 登录钉钉 -> 点击`签到`跳转到签到界面

**注意：只能用钉钉sso登录，若用学号和密码登录无法正常签到**

### 日常签到

与初次使用类似，点击`登录`再点击`签到`。 **要先点登录，直接签到很可能会话会过期。** 钉钉大概会记住你的账号，点一下`立即登录`即可。若你发现还是需要登录钉钉，请查看后文`常见问题`

### 退出登录 & 更换账号

点击`清除`按钮后点击`登录`重新使用钉钉登录
